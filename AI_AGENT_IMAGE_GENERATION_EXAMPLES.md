## üîÑ Image Workflow

### Complete Process

```
1. WRITE the blog post content

2. SELECT the appropriate prompt from this guide based on:
   - Post category (Credit, Funding, Business)
   - Post topic (specific subject matter)
   - Visual theme needed

3. GENERATE the image using your AI tool:
   - DALL¬∑E 3: Best quality, most realistic
   - Midjourney: Creative, artistic control
   - Stable Diffusion: Fast, customizable

4. VERIFY the image meets requirements:
   - 16:9 aspect ratio (1200x675px minimum)
   - Professional, business-appropriate
   - No visible AI artifacts or distortions
   - Under 500KB file size

5. UPLOAD to hosting service:
   - Imgur: https://imgur.com/upload (easiest)
   - Cloudinary: https://cloudinary.com (professional CDN)
   - GitHub: /public/blog-images/ (version controlled)

6. GET the direct image URL:
   - Must end in .jpg, .png, or .webp
   - Example: https://i.imgur.com/abc123.jpg

7. ADD to blog post JSON:
   {
     "image": "https://i.imgur.com/abc123.jpg"
   }

8. PUBLISH to GitHub using standard workflow
```
